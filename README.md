# Sensation Utilities Plugin
A plugin that has many utilities needed for today's Roblox.

## Installation
The main place to install this plugin is from the Roblox Marketplace [**module**](create.roblox.com/store/asset/74082080143456). To otherwise install this plugin locally use the information provided below.

### Linux
Steps to install locally:
1. Download [**loader.luau**](https://github.com/Adi0nt/Sensation-Utilities-Plugin/raw/refs/heads/main/plugin.luau)
2. Locate the plugins folder by opening **Roblox Studio**, pressing the **plugins** ribbon at the top left of the application, and pressing on **Open Plugins Folder**.
3. Move **loader.luau** into the plugins folder you have just opened.
4. Restart studio and use.

### Mac
Steps to install locally:
1. Download [**loader.luau**](https://github.com/Adi0nt/Sensation-Utilities-Plugin/raw/refs/heads/main/plugin.luau)
2. Locate the plugins folder by opening **Roblox Studio**, pressing the **plugins** button at the top left of your screen, and pressing on **Open Plugins Folder**.
3. Move **loader.luau** into the plugins folder you have just opened.
4. Restart studio and use.

### Windows

## Troubleshooting
To help fix this plugin for yourself use the information provided below.

### No plugin icon in plugins tab
Restart all studio instances if its missing from the plugins tab.

### Require error
Use the local loader on the github following the instructions above.
